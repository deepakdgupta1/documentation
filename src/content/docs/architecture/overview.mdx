## Components Description

- **MDX/Markdown Content**: Hand-written documentation and architectural records.
- **openapi.yaml Spec**: Single source of truth for the API routes.
- **starlight-openapi**: Auto-generates the reference documentation pages.
- **starlight-llms-txt**: Compiles plain text mappings for AI models at build time.
